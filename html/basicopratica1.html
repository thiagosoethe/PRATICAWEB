Tags de cabeçalho h1 - h6

h5 - Tag de cabeçalho de mais alto nivel de importancia dabaixo a tab "h4"
h6 - Tag de cabeçalho de nivel de importancia baixo a tab "h5"

Escrevendo um parágrafo

Esta Tag define um paragrafo para um texto escrito em uma pagina
HTML podendo este conter apenas um ou mais linhas.
Perceba também que utilizamos a tag "br" antes do parágrafo, para pular linha do trecho anterior para este e também na produção do texto para pular as linhas.
Há, um outro detalhe, veja que utilizamos a tag "strong" para grifar em negrito
